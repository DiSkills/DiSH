# DiSH is a simple command interpreter

### Features:
  - [x] Input of commands
  - [x] Output arguments in the format: \[argument]

#### Special chars:
  * " - toggles the argument splitting mode (text inside is one argument)
  * \\ - escapes the next char (" or \\)

#### Errors:
  * Unmatched quotes
  * Unsupported escape sequence
