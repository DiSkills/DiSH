# DiSH is a simple command interpreter

### Features:
  - [x] Input of commands
  - [x] Output words from the command in the format: \[word]

#### Special chars:
  * " - toggles the reading mode for whitespace chars
  * \\ - escapes the next char (" or \\)

#### Errors:
  * Unmatched quotes
  * Unsupported escape sequence
